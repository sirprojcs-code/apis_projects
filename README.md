# Project 1: Facebook Graph API Viewer
# Description
This project allows users to view and search Facebook posts from a specific page or user using the Facebook Graph API. Users can input the Facebook page ID and optionally filter posts by keywords. The app displays posts in card format with post messages, dates, and engagement stats.
# Features
•	Search posts by keyword.

•	Fetch and display posts with messages, dates, likes, and comments.

•	Loading spinner while fetching data.

•	Error handling for invalid page IDs, permissions, or network issues.

•	Responsive and modern UI design.

# How to Run
1.	Clone or download the repository.
	
2.	Open the folder Facebook Graph API Viewer.
	
3.	Open index.html in your browser.


# Project 2: Google Calendar API Viewer
# Description
This project allows users to view upcoming events from their Google Calendar and create new events directly through the interface using the Google Calendar API. Users can filter events by date range or search keywords, and they can create events using an OAuth access token.
# Features
•	Load events from a specific calendar.

•	Search events by keyword.

•	Filter events by start and end date.

•	Create new events with title, date, time, and optional description.

•	Loading spinner and proper error messages.

•	Responsive and clean interface.

# How to Run
1.	Clone or download the repository.
	
2.	Open the folder Google Calendar API Viewer.
	
3.	Open index.html in your browser.
